Bucza-Kube
==========

After few hours fight with minikube i decide to create k8s cluster from scratch.

Toolchain
---------

+ KVM
+ libvirt
+ virt-install
+ kickstart


Usage
-----

Before start run local http server for files example:

```
python3 -m http.server
```
and then in other shell

```
sudo bash bootstrap.sh
```
